手机摄影指南

发表于 2020-01-21

怎样用手机拍好人像

三个要点：机位、色彩和调动

机位决定对不对

  • 全身照：机位在腰或腰以下，拍出大长腿就对了
  • 半身照：机位靠上，从上往下拍显得脸小。半身照禁忌，有部分身体被截在画面外，千万不要截在人的关节部位，如膝盖、手肘、关节等部位
  • 面部特写：机位在眼睛。因为焦点在眼睛，所以机位也要在眼睛的位置。

其他技巧：

  1. 补光神器：A4纸

  2. 如果画面单调可借助自身的元素，如手，可以让被拍摄对象不停地变化手部的姿势和面部做一个配合，比如手放在额头上、脸颊上或者嘴边，或者双手合十放在脸庞，手臂交叉、摸肩膀，下巴靠在手腕上等等。

  3. 借助道具

    水杯。女生拿水杯，倾向于拍出柔美和可爱；男生拿水杯，可以拍出很深层的一面

    小盆栽：当前景用。让叶子最大限度的贴近镜头，在画面中形成光晕和色块，人为制造景深。还可以用这些小道具来遮挡环境中的混乱，让主体更突出。

    大盆栽：当背景用。让画面活泼起来，还能在室内拍出室外的感觉

  4. 高级玩法：借助玻璃。隔着玻璃拍人，多试几个角度，会得到出乎意料的光影效果。玻璃上还可以洒水

色彩决定好不好

色彩这件事说起来特别主观，但色彩是有功能的，它能帮助表达情绪。

  1. 如果想突出一个人的面部表情，突出他的情绪，建议让画面的颜色尽量少些，或者尽量同一个色系。

  2. 当要人的状态显得很有动感的时候，需要选择对比色，如红和绿，黄和蓝

  3. 背景非常鲜亮时，可以让被摄对象摆出有动感的、大幅度的姿势，充分彰显人物的个性

  4. 黑白处理，突出情绪，能够稀释掉色彩带来的干扰，特别适合在人物特写的时候表达他的状态

调动你的被拍对象

姿势太僵硬，表情不自然，怎么办？不经意的刻意，摆拍中有抓拍

  1. 拍摄对象基本都是熟人，可以从特写开始，由近及远地去调用他的情绪，在这个过程中，不断的跟他说笑，大笑，特别好，特别对等等。通过这些反馈让对方建立自信，然后你会发现对方会慢慢放开，会主动摆pose给你拍，这个过程中，快门不能停，这样可以拍到很多情绪自然的好照片
  2. 有些人怵镜头，不自然。应对方法是别让她看镜头，引导她看窗外，看地面，或者看你背后的某一个点
  3. 如何快速调动被拍对象：核心六个字，“低头，伸下巴，笑”

如何拍好很难调动的小孩

积极互动，守株待兔

第一步，选一个光线充足的空间或角度

第二步，认真和孩子玩耍，准备好手机。过程中不能开闪光灯

第三步，使用手机连拍功能

怎样用手机拍好风景

把风景拍「对」的三个原则：横平竖直、控制曝光和手要稳

  • 横平竖直：打开手机中的辅助线，利用辅助线和地平线或者建筑上的直线做一个找准
  • 控制曝光：看手机屏幕上焦点的方框旁边有个小太阳，按住它往下拉，稍微把曝光降低一些，即使画面黑了一些也没关系，可以利用后期调回来。如果画面很亮，丢失的细节是调不回来的。然后用你的手遮住镜头的上方，形成一个遮光板的效果，避免杂光进入画面
  • 手要稳：可以用两只手夹住手机，用音量键当快门。另外，每次按完快门，等半秒再放下手机，防止画面变虚

把风景拍「好」的三点技巧：

  1. 风景放在早晚拍：正午12点，光线强烈，很容易让照片过度曝光，看上去白晃晃，没有明暗冲突，缺少立体感。日出之后的两三个小时和日落之前的两三个小时内都非常适合拍照。暖暖的光线从很低的角度照在建筑物的侧面，刚好和天空冷冷的色调形成对比，显得特立体。这种光被称作侧逆光，无论是顺光拍还是逆光拍，出来的效果都不错
  2. 利用好自然线条：如对树进行仰拍，虽然其他树枝会造成线条乱，这时我们可以用黑白来去除干扰，这样拍出的照片就很有生命力；对于很有线条感的建筑物，可以从对角线把画面分成两部分，人为突出线条的秩序感；利用雨雪天气，其实会给画面增加线条和趣味
  3. 构图给景观加戏：找到前景，人为制造景深。利用环境给景观加一个画框

如何拍好风景中的人：避免把旅游纪念照拍成到此一游

  • 大风光，突风景：如果是大风光，如大草原，山顶公路上，建议突出景观的辽阔。操作方式是尽量采用横构图，把拍摄角度放低；拍背影也是个不错的选择，能迅速提升格调。这样拍出来一般是4:3的画幅，可以裁剪成16:9的比例，这是电影屏幕的比例，会让照片自带故事感
  • 小风景，突出人：人和景的大小该如何安排，可以用万能的构图法—-三分法，不管全身还是半身照，不管横版还是竖版，几乎都可以用三分法，而手机上的辅助线刚好是三分法，可以利用它来保障照片的横平竖直，拍风景中的人时，把人物放在一条线上，留出一边给环境，如果能把人的头部放在横竖两条线的交点上，这样拍出来照片怎么看都和谐。还可以利用画框法

提醒：时间紧张的情况下，不用在乎上面的技巧，在框定要拍的风景之后，横竖各拍一张，之后再利用后期软件去增加效果，不过要注意的是不要变焦，一旦变焦，往往1000万的像素就变成500万，画质损失大

怎样用手机拍好静物

拍好食物的一条重要辅助线，就是你看到这张照片，你有没有食欲，食物真实有细节特别重要

  • 一个原则：细节制胜

  • 增加细节的技巧：补光。拍小物件时，光源往往来自于上方，会让物体的上面很亮，侧面很暗。此时可以用另外一只手机打开手电筒，从侧面给光，注意距离，不要补光过度；还有一个办法是用A4纸，把纸立在静物的暗面，光被反射回来会变得非常柔和

极简手机修图指南

修图原则：还原真实，突出细节，强化细节，强化情绪

旅行摄影指南

自然感，高级感，电影感

旅行就是去不同的地方，感受不同的文化，享受不一样的快乐。而旅行摄影是为了记录旅行这个过程，记录旅行就是记录时间，就是记录自己的人生。所以，好的旅行照片就是在用有自然感，高级感，电影感的照片在写一篇旅行日记。

准备篇

设备 主设备手机，其他配件：自拍架、八爪鱼、补光灯
服装穿搭 百搭款「白上衣+休闲裤/牛仔裤，女生可以穿白裙子」
风格搭配 衣服与场景协调
色彩搭配 核心原则「身上最好不要超过3种颜色」
做功课 提前摸清目的地主色调，背景颜色比较杂，推荐黑白灰；背景为简单色，搭配同色系，也可搭配对比色。还可以准备一些其他小饰品

人多怎么办？

  • 技巧一：早起早到，不仅人少，光线还好
  • 技巧二：调整角度，仰拍人物和地标
  • 技巧三：iPhone广角效果
  • 技巧四：长曝光

姿势篇

拍摄者 积极与被拍摄者互动
被摄者 要不要看镜头？自然就看，不自然就不看。在自然的状态下，和周围去互动
初级互动 有什么，就和什么互动，比如说有面墙，那就去靠着墙
中级互动 借助道具，让画面变得更有趣，如戴着帽子等
高级互动 和环境互动，比如说你可以从画面的左、中、右、前、中、后，走起来，然后抓拍一张。还可以用连拍，设置录像后截屏
合照的要点 错位。不要让所有人在同一个平面，要有一个前中后的错位

构图篇

旅行摄影构图原则：人小景大

  1. 设计画面引导线：「引导线」是指具有明显方向性的线条
  2. 在景色中寻找自然的框架构图，构图时注意对边框的对齐平直
  3. 利用前景遮挡，如透过叶子和花去拍远处的风景，让画面更有层次感
  4. 运用留白，要留出干净的背景，这种构图方式不会分散别人的注意力

光线篇

  1. 阴天适合拍人像。因为阴天是柔和的光线，每个角度都能拍，还能保留很多细节

  2. 日出日落的前后半小时拍户外。影像界的黄金时段，漫射效应最明显

  3. 中午拍室内。不要让脸正面迎着光，稍微侧一点。如果在户外拍摄阳光比较充足，明暗反差大的地方可以打开HDR模式

  4. 逆光拍剪影。

    • 首先确保拍摄对象是完全背光的，正面最好不要有任何补光的东西出现；
    • 第二个是降低曝光，拍剪影的画面重点是背景，所以在对焦之后要拉低画面的曝光度，直到后面的背景达到合适的饱和度再按快门
    • 第三点是要有好看的肢体语言，无论是静止还是运动，会给人一种强烈的冲击力和意境

  5. 夜景。要找到所处环境最亮的地方，让人的脸部尽量靠近最亮的部分去拍摄，背景最好有足够的空间和明暗关系

自拍篇

  1. 侧面不看镜头，留空间给景色
  2. 后置摄像头借助玻璃或镜子自拍
  3. 录像+截屏

修图篇

旅行照片修图的两个动作：

  1. 提高反差
  2. 增加锐度
