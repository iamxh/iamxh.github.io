给人好印象的秘诀

发表于 2019-05-02

一、主观感受的形成机理和步骤

  1. 根据外表、肢体动作、言语、地位等透露出的信息，无意识、自动地对你形成初步印象(人们都是认知吝啬鬼)
  2. 如果观察者发现之前的判断有误，会启动纠偏机制。这个过程费时费力，如果没有强烈的动机，一般不会主动进入

二、给人留下好印象的三个关键点

  • 信任透镜、自我透镜、权势透镜。
信任透镜及应对方法：

别人首先要确认你是否值得信任，是敌是友。别人对你的看法是负面还是正面，90%取决于你表现出来的温暖程度和个人能力。

  1. 提高温暖程度

    • 专注倾听。保持眼神交流、点头和微笑
    • 表现同理心。关键是换位思考，尝试寻求双方的共性
    • 首先信任别人。比如和他分享遇到过的困难，犯过的错误。

  2. 展现个人能力

    • 显示你的意志力。表明你能抵抗各种诱惑和利益
    • 适当谦虚。表现谦虚时，别人在评估你真实能力时会额外增加20%到30%的估值。自吹自擂会得到相反的结果
    • 适当摆出些有权势的姿态
    • 多强调你的潜力
自我透镜及应对方法：

保护和增强观察者的自尊，让观察者感觉自己更出类拔萃。

观察者与别人比较时，会看别人的成就与自己的相关性。如果相关性高，对观察者就是高度威胁，为了维持自尊，对方将不得不与你保持更大的距离，或者选择忽视你。

  1. 示弱。有意降低你对观察者自尊的威胁
  2. 肯定对方。把注意力集中在对方最擅长的方面，增强其自信心
  3. 寻找共性。营造”咱们是一伙的”感觉
权势透镜及应对方法：

给相对”有权势”的人来评估你对他是否”有用”。这里说的权势是中性词，也就是主动权。权势是动态的和变换的。

有权势者对你的依赖性更小，在潜意识里更倾向于走捷径，使用各种偏见、成见和假设，用最小精力观察你。比如对老板的换位思考，应该在他给你布置任务前，预测他的需要，并提前做好准备。
