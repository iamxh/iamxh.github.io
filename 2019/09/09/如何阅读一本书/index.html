如何阅读一本书

发表于 2019-09-09 更新于 2019-09-17

读书的四个层次：

  1. 基础阅读：能认字，能明白一句话的意思，就算是会基础阅读了
  2. 检视阅读：在很短的时间内，快速地、粗略地、大概地读一下这本书
  3. 分析阅读：学会怎么列一本书的大纲，怎么理解重点词，找中心思想，理解一本书的核心观点
  4. 主题阅读：围绕一个主题读好几本，甚至几十本

另外，读书快慢不是重点，没有绝对的好坏，要根据读书的目的来调整。而针对不同类型的书，也有不同的阅读方法。

检视阅读

其实就是”略读“、”速读“、”粗读“的意思，其作用就是帮你挑书

挑书的六个步骤：

  1. 看书名言、前言：明确了主题和类型，类似读后感
  2. 看目录：类似看预告片
  3. 看索引：并不是每本书都有，通过索引里的关键词，猜这本书的内容
  4. 看宣传语：虽然是广告，但还是能够告诉你本书的什么亮点值得你关注
  5. 看重点段落
  6. 随便翻几页

怎么读完一本有难度的书？

方法就是从头到尾把书读一遍，跳过所有你读不懂的部分。

如果你一碰到不懂的地方就停下来，反倒容易越读越糊涂，但如果你用检视阅读的方法，跳着把书读完，你至少能有两方面的收获。

第一，你最开始读不明白的东西，读到后面居然渐渐清楚了。

第二，你能享受到读完整本书的成就感。

提升阅读速度的小技巧

那就是让眼睛看书的时候动起来，跟着手指读书。

分析阅读

分析阅读就是细细读一本书，分析一本书，直到你把这本书都读明白了。

“分析阅读”的五个招法：

  1. 用一句话概括一本书

    • 并不是真的只有一句话，而是强调尽量简单
    • 本质是找到一本书的重点

    如何找一本书的重点？有三个方法

    1. 通过书的类型来找重点，所有的书可分为两类：虚构类的小说和“论说类”。

      如果是虚构类的小说，那你找重点，就要找重要角色，重要情节。如果是论说类的书，那这些书的重点，往往就是作者给你讲了哪些具体的知识。

    2. 从书名里找重点。

    3. 从作者的写作目的来找重点。

  2. 列大纲

    就像切蛋糕一样，你可以先把一本书分成几大块，每大块再切成几个小块。

  3. 理解关键词

    书里面那些反复出现的词，还有那些不弄明白书就读不下去的词，都是关键词。

  4. 找到关键句

    一般来说，关键词扎堆出现的地方，很有可能就是一本书的关键句。

  5. 怎么赞同或者反对一本书

    做一个礼貌的读者，你要完完整整地读完一本书，读懂一本书之后，再提出赞同或者反对的评价。

主题阅读

就是同一个主题的书你读上好多本，然后再总结出这个主题相关的几大重点。

怎么读一本小说

  1. 先一口气读完剧情
  2. 体会小说里的细节
  3. 进入小说世界里

怎么读诗歌和历史

诗歌

  1. 大声读出来
  2. 关注诗歌的主题

历史

  1. “想想为什么”
  2. “多读几本”
