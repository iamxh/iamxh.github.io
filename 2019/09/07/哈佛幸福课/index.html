四个汉堡模型

为了期末考试，努力学习了好久，考完试之后，你打算去一家餐厅吃汉堡，来犒劳犒劳自己。你有四种选择

  1. 一口气吃三个招聘汉堡，这种汉堡非常好吃，但不太健康，吃多了容易拉肚子
  2. 吃一个素汉堡，这种汉堡不怎么好吃，但很健康
  3. 一个汉堡都不吃。因为你担心自己考不好，不想庆祝
  4. 吃一个好吃的招牌汉堡

四个汉堡分别对应着四种看待幸福的态度，分别是“享乐主义”、“忙碌奔波”、“虚无主义”和“感悟幸福”

  • 享乐主义

    就是沉浸在当下的物质享受当中，并且把这种享受当成快乐的唯一来源。这种心理并不会让你体会到真正的幸福，因为物质一旦过度，就不会让人快乐。

    享乐主义过于注重物质享受，但物质享受带来的快乐是短暂的、不可持续的。

    就像中奖的人，在狂喜一阵后，最后还是会回归之前平静的状态。

  • 忙碌奔波型

    和享乐主义刚好相反。享乐主义强调物质享受和当下的快乐，而忙碌奔波型的人，是放弃了当下的快乐，把追求长远目标，当成唯一的动力。

    在平时学习生活当中，你也许对自己要求相当高，而且为了达到目标相当地刻苦。但如果你在这个过程中，感受到的不是快乐和充实，那从“积极心理学”的角度来看来，这样的心理状态也是不好的

  • 虚无主义

    积极心理学的开山鼻祖叫塞里格曼曾经做过一个实验，把三组小狗分别放进三个房间

    房间一：小狗会被电击，但小狗身边有个开关。只要小狗碰到这个开关，电击就会停止

    房间二：小狗也会被电击，但不管它做什么，电击都不会停止

    房间三：小狗不会被电击

    接着把这三只小狗，都放进一个大箱子里。这大箱子四周，有低矮的栅栏，只要小狗轻轻一跳，就能跳出这个箱子。这时候，心理学家开始对三组小狗都进行电击。

    实验结果发现，第一组和第三组的小狗，都很快地跳出了箱子。就只有第二组的小狗，没有跳出箱子

    偶尔的虚无主义倾向是有办法可以解决的，就是用具体事实说服自己，失败并不可怕。

    面对挫折的方法：

    第一，这次失败会有什么具体后果？

    第二，这次失败有哪些具体的原因？

    觉得自己走不出失败，对未来失去信心时，可以看看这两个问题，试着回答。回答时要避免用“永远”、“一直”、“再也不”、“一定”这样的词。回答一定要是事实

  • 感悟幸福

    如果说享乐主义是快乐太多，忙碌奔波是没有快乐，虚无主义是焦虑无助，这些都不算是积极的心态的话，那怎样的心理才能够通往幸福呢？

    就是既拥有了当下的享受和快乐，又没有放弃对目标和意义的追求。要获得幸福，就是要把当下的快乐和长远的意义结合起来。

幸福就是“一块钱”

幸福公式，也叫做MPS公式，掌握了这个公式，就掌握了一种培养积极情绪、通向幸福的方法。

MPS：M代表meaning，意义；P代表pleasure，快乐；S代表strength，潜力、优势。也就是意义、快乐加潜力，简称“一块钱”

如何运用“一块钱”公式？

第一步，回答自己一个问题，什么事情对我来说是有意义的？

第二步，培养让你快乐的小习惯

第三步，发现你的潜力，同时接纳不完美的你

积极心理学把优秀的性格分为六大类：

  1. 智慧和知识：就是说你喜欢接触新鲜东西，愿意学习新的东西，思考一件事情比较全面，或者愿意接受和平常不一样的事。
  2. 勇气：你愿意说实话，开始做一件事情就不喜欢放弃，或者遇到困难、挑战的时候想要去克服。
  3. 人道主义：是说你愿意跟人打交道，珍惜和别人的感情和关系，或者能够体会到别人的感受和情绪。
  4. 公正：是说你能客观地评价别人和自己，或者你可以在团队里扮演一种角色，无论是当领导还是当普通成员。
  5. 节制：是说你能原谅别人的错误，为人比较谦虚低调，做事情比较小心谨慎，或者能够控制自己的情绪或行为。
  6. 卓越：是说你能时时感受到激情，发现美或别人的优点，为人幽默，能够表达感激，或者心中怀抱希望。

每个人都有值得自己骄傲的长处，你要学着发现它们、珍惜它们。

防御性悲观

害怕、焦虑的根源是什么？其中最重要的一个原因是人有联想未来的能力。

在一个风雨交加、电闪雷鸣、伸手不见五指的——大白天，你一个人走到茫茫的沙漠里，头上阳光非常毒辣，这时候，你觉得非常地——冷

这段话你可能会说毫无逻辑，所以你看，你会本能的联想未来，这就叫“想象力”。

人会为了想象当中的幸福，去推迟梦想成真的时刻。

人为什么要联想未来？

常常是为了计划和面对未来。在联想未来时，总是由些负面的，因为我们启动了“防御性悲观”的心理机制，这个机制会让我们高估未来的困难，产生一定的焦虑感。这么做的好处是会激励我们做好充足的准备，来应对未来可能会遇到的困难。

人之所以喜欢计划未来，是因为人喜欢掌控未来的感觉。掌控未来的感觉，会让人更幸福。

你的大脑是个骗子

第一组，阳光，沙滩，海浪，海鸥，椰树，小岛。

第二组，白光，雪地，雪山，围巾，雪糕，雪花。

第三组，雨伞，小巷，细雨，青苔，白墙，黑瓦。

好，现在请你睁开眼睛，回忆一下，刚刚的第一组词里，有“海岛”这个词吗？

因为你的大脑在储存信息的时候，不会原封不动地把所有信息都存下来。

所以你的大脑是个骗子，会让你想象出不准确的未来。具体原因有两点。

第一，想象的未来是缺少细节的。

第二，想象的未来，会被当下的情景束缚。

所以，想象的未来常常是不准确的。当未来没你想象得那么好的时候，你就会失望、难受；但如果未来没你想象得那么糟，你又会白白痛苦了好久。

所以说，不论是快乐的联想，还是痛苦的联想，都会给你带来没有必要的焦虑。如果咱们想要过得更加幸福，就应该试着少联想未来，多关注现在。

心理免疫系统

  1. “心理免疫系统”，在打击和痛苦来临时，它会帮助你调节心态，让你更好的面对打击，承受痛苦。
  2. 在大的打击来临时，你的“心理免疫系统”会被激发，让你主动地去面对痛苦。而在一些小的打击或挫折面前，“心理免疫系统”就不会被激发。所以人在遇到大的痛苦时，往往能更积极地去面对。
  3. 在没有选择的时候，人的“心理免疫系统”往往会选择接受当下的结果，这样反而能够获得内心的满足和幸福。【没选择比有选择更幸福】

如何做才能更幸福

两个获得幸福的方法：

第一个方法是：停止想象，开口去问。

第二个方法是，停止想象，把握当下。
